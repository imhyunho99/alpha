## 🎯 투자 전략

### 전략 개요
- **매월 초** 단기/중기/장기 Top 10 추천을 각각 받음
- **총 30개 자산**에 균등 분산 투자 (각 자산 3.33%)
- **단기/중기/장기 비중**: 1/3씩 (각 33.33%)
- **월간 리밸런싱**으로 포트폴리오 재구성

### 스코어링 기준
- **단기 (1주)**: RSI 모멘텀 70% + AI 예측 30%
- **중기 (3개월)**: 이동평균 추세 80% + 단기 점수 20%
- **장기 (1년)**: 연간 수익률 60% + 안정성 40%

## ⚠️ 백테스팅 제약사항

### 데이터 한계
현재 시스템의 데이터 구조상 다음과 같은 제약이 있습니다:

1. **미래 데이터 포함**: 현재 DB에 2025-2026년 데이터가 포함되어 있어 순수한 과거 백테스팅이 불가능
2. **스코어링 엔진 의존성**: 과거 시점의 스코어를 재현하려면 해당 시점의 모델 상태가 필요
3. **Look-ahead Bias**: 현재 모델은 최신 데이터로 학습되어 과거 예측 시 미래 정보가 반영됨

### 권장 백테스팅 방법
정확한 백테스팅을 위해서는:
- 과거 각 시점마다 모델을 재학습
- 해당 시점까지의 데이터만 사용
- Walk-forward 방식의 검증 필요

## 🎯 전략의 강점

### 1. 동적 자산 선별
- 200+ 자산 중 매월 상위 30개 선택
- 시장 상황에 따라 포트폴리오 구성 변경
- 저성과 자산 자동 제외

### 2. 다층 시간대 분석
- 단기: 모멘텀 포착
- 중기: 추세 추종
- 장기: 안정적 성장
- 각 시간대별 최적 자산 혼합

### 3. AI + 기술적 분석 융합
- 머신러닝 예측 + 전통적 지표
- 과적합 방지 (200+ 개별 모델)
- 지속적 학습 가능

### 4. 리스크 관리
- 30개 자산 분산
- 월간 리밸런싱
- 자산군 다각화 (주식 + 암호화폐)

## ⚠️ 전략의 위험 요소

### 1. 모델 리스크
- AI 예측 정확도 한계
- 과거 패턴이 미래에 반복되지 않을 수 있음
- 블랙스완 이벤트 대응 어려움

### 2. 시장 리스크
- 암호화폐 높은 변동성
- 시장 전체 하락 시 손실 불가피
- 유동성 리스크

### 3. 운영 리스크
- 거래 비용 및 슬리피지
- 리밸런싱 타이밍
- 데이터 품질 의존성

### 4. 기술적 리스크
- 모델 업데이트 필요
- 데이터 수집 중단 가능성
- 시스템 장애

### 핵심 요약

Alpha 투자 전략은 **AI 기반 자산 선별**과 **다층 시간대 분석**을 결합한 체계적인 접근 방식입니다.
