# Alpha v2.0: AI 기반 투자 분석 및 추천 시스템

Alpha는 동적 자산 스크리닝, AI 예측, 투자 가치 스코어링을 결합하여 정량적 투자 추천 및 포트폴리오 분석을 제공하는 클라이언트-서버 애플리케이션입니다.

## 아키텍처

이 프로젝트는 두 가지 주요 구성요소로 이루어져 있습니다.

1.  **Alpha 서버 (`alpha_server/`):**
    *   Python `FastAPI`로 구축된 백엔드 API 서버입니다.
    *   **동적 자산 스크리닝:** NASDAQ 100, Top 100 암호화폐 목록을 동적으로 수집하여 분석 대상을 확장합니다.
    *   **데이터/모델 파이프라인:** 수백 개 자산의 시세 데이터를 수집/저장하고, 각 자산에 대한 AI 예측 모델을 자동으로 학습시킵니다.
    *   **스코어링 엔진:** AI 예측과 여러 기술적 지표를 조합하여, 자산별 '투자 가치 점수'를 단기/중기/장기 관점으로 산출합니다.
    *   **API 제공:** 클라이언트에게 Top-N 투자 추천, 포트폴리오 상세 분석 등 정량적 분석 결과를 API 형태로 제공합니다.

2.  **Alpha 클라이언트 (`Alpha/`):**
    *   Python `PySide6`로 구축된 데스크톱 GUI 애플리케이션입니다.
    *   사용자가 투자 기간(단기/중기/장기)을 선택하고, 서버로부터 'Top 10 투자 추천'을 받아볼 수 있는 직관적인 인터페이스를 제공합니다.
    *   포트폴리오 파일을 업로드하면, 각 자산의 수익률과 함께 기간별 투자 가치 점수를 상세히 비교 분석해줍니다.

## 전체 시스템 실행 방법

클라이언트와 서버는 각각 별도의 가상 환경과 터미널에서 실행되어야 합니다.

### **1단계: 서버 실행**

1.  **새 터미널**을 엽니다.
2.  프로젝트의 루트(`pythonWorkspace`)에서 다음 명령어로 서버의 가상 환경을 활성화하고 Uvicorn 서버를 시작합니다.
    ```bash
    # (프로젝트 루트 디렉토리에서 실행)
    alpha_server/venv/bin/uvicorn alpha_server.main:app --reload
    ```
3.  터미널에 `Uvicorn running on http://127.0.0.1:8000` 메시지가 나타나면 서버가 정상적으로 실행된 것입니다. 이 터미널은 계속 열어두세요.

### **2단계: 클라이언트 실행**

1.  **또 다른 새 터미널**을 엽니다.
2.  프로젝트의 `Alpha` 디렉토리로 이동하여 가상 환경을 활성화합니다.
    ```bash
    cd Alpha
    source venv/bin/activate
    ```
3.  GUI 클라이언트를 실행합니다.
    ```bash
    python -m alpha.main --gui
    ```
4.  GUI 창이 나타나면, 먼저 서버의 데이터와 모델을 준비하기 위해 **"서버 데이터 업데이트 요청"**과 **"서버 모델 재학습 요청"** 버튼을 차례로 누릅니다. (이 과정은 시간이 다소 소요될 수 있습니다.) 그 후 "Top 10 투자 추천 받기" 등의 분석 기능을 사용할 수 있습니다.

## 주요 기능 (클라이언트 GUI v2.0 기준)

*   **서버 제어:**
    *   `서버 데이터 업데이트 요청`: 수백 개 자산의 최신 시세 데이터를 서버가 다운로드하도록 요청합니다.
    *   `서버 모델 재학습 요청`: 모든 자산에 대한 AI 모델을 서버가 다시 학습하도록 요청합니다.
*   **분석 기능:**
    *   `Top 10 투자 추천 받기`: 사용자가 선택한 투자 기간(단기/중기/장기)에 대해 서버의 스코어링 엔진이 계산한 '투자 가치 점수' 상위 10개 자산 목록을 받습니다.
    *   `포트폴리오 평가`: 사용자의 포트폴리오를 업로드하면, 각 자산의 수익률과 함께 단기/중기/장기 투자 매력도 점수를 비교 분석하여 보여줍니다.
