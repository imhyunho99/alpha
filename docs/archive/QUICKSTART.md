# Alpha 프로젝트 빠른 시작 가이드

## ✅ 시스템 정상 작동 확인 완료!

모든 핵심 컴포넌트가 정상적으로 작동합니다:
- ✅ 서버 모듈 (FastAPI)
- ✅ 클라이언트 모듈 (GUI)
- ✅ 데이터 핸들러 (QuestDB/CSV)
- ✅ AI 모델 핸들러
- ✅ 자산 스크리너

## 🚀 3단계로 시작하기

### 1단계: 서버 시작

터미널을 열고 다음 명령어를 실행하세요:

```bash
cd /Users/nahyeonho/pythonWorkspace/Alpha
./start_server.sh
```

다음과 같은 메시지가 표시되면 성공입니다:
```
INFO:     Uvicorn running on http://127.0.0.1:8000
INFO:     Application startup complete.
✅ 자동 데이터 업데이트 활성화 (6시간마다)
```

### 2단계: GUI 클라이언트 시작

**새 터미널**을 열고 다음 명령어를 실행하세요:

```bash
cd /Users/nahyeonho/pythonWorkspace/Alpha
./start_gui.sh
```

GUI 창이 열립니다.

### 3단계: 첫 사용 설정

GUI에서 다음 순서로 버튼을 클릭하세요:

1. **"서버 상태 확인"** - 서버 연결 확인
2. **"서버 데이터 업데이트 요청"** - 시장 데이터 다운로드 (5-10분 소요)
3. **"서버 모델 재학습 요청"** - AI 모델 학습 (10-20분 소요)

이제 **"추천 받기"** 버튼으로 투자 추천을 받을 수 있습니다!

## 📊 주요 기능

### 투자 추천 받기
- **단기**: 1주일 이내 투자 (모멘텀 중심)
- **중기**: 3개월 이내 투자 (추세 중심)
- **장기**: 1년 이상 투자 (수익률 & 안정성 중심)

### 포트폴리오 분석
보유 자산의 성과와 투자 가치를 평가합니다.

## 🔍 API 문서 확인

서버가 실행 중일 때 브라우저에서 다음 주소를 방문하세요:

```
http://127.0.0.1:8000/docs
```

대화형 API 문서를 통해 모든 엔드포인트를 테스트할 수 있습니다.

## 💡 팁

### 데이터 업데이트 주기
- 서버는 6시간마다 자동으로 데이터를 업데이트합니다
- 수동 업데이트가 필요하면 GUI에서 버튼을 클릭하세요

### 성능 최적화
- 처음 실행 시 모든 자산(200개 이상)의 데이터를 다운로드하므로 시간이 걸립니다
- 이후에는 증분 업데이트만 수행되어 빠릅니다

### 벤치마크 모드
전체 자산 대신 5개 벤치마크 자산만 사용하려면:
```python
# alpha_server/asset_screener.py에서
# get_all_tickers() 대신 get_benchmark_tickers() 사용
```
