# Alpha 시작 가이드

## 🚀 한 번에 실행하기

### macOS / Linux
```bash
./start_alpha.sh
```

### Windows
```cmd
start_alpha.bat
```

이 명령어 하나로:
1. 기존 서버 자동 종료
2. 서버 백그라운드 실행
3. GUI 자동 실행
4. GUI 종료 시 서버도 자동 종료

## 📝 첫 실행 시 해야 할 일

GUI가 열리면:
1. **"서버 데이터 업데이트 요청"** 클릭 (5-10분)
2. **"서버 모델 재학습 요청"** 클릭 (10-20분)
3. **"추천 받기"** 클릭

## 🎯 그게 끝입니다!

더 이상 복잡한 과정이 없습니다.
